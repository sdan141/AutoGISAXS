## General Information
- In situ grazing incidence small angle X-ray scattering (GISAXS) is a powerful tool for accessing nanoscale structure formation in real time with high time resolution and high statistical relevance. 
Nevertheless, the analysis is clearly time consuming and challenging task, necessitating the need for strategies to speed up the process.
- Among other techniques, the use of neural networks (NN) was proposed.
- So far, there is little labeled experimental data, so the use of simulations for training the learning algorithm has been proposed. It has been shown that despite the limitation of idealized geometries, there is a good agreement with the data if the main features are taken into account.
- Previous approaches aimed at predicting a single numerical value using regression models were unsuccessful.
- A recently developed approach, in which the distribution of predictions is determined from a probability vector, provides more promising results.
- The aim of this project is to make the developed method adaptable for the users of the facility so that they can try out several models and check which one is best suited to their specific requirements and apply this model to new experimental data.
- In other words, the evaluation time should be shortened and study results can be obtained much more easily and quickly.
